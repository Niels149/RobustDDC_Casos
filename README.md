# RobustDDC_Casos

This project implements a robust data-driven control program based on the following paper utilizing the [Casos](https://github.com/iFR-OFC/casos) tool for controller and observer synthesis.

**Reference:** 
J. Zheng, T. Dai, J. Miller, et al., "Robust Data-Driven Safe Control Using Density Functions," IEEE Control Systems Letters, vol. 7, pp. 2611–2616, 2023.

T. Cunis and J. Olucak, ‘CaΣoS: A nonlinear sum-of-squares optimization suite’, in 2025 American Control Conference, (Denver, CO), pp. 1659–1666, 2025. doi: 10.23919/ACC63710.2025.11107794

**Original Implementation:** [GitHub Repository](https://github.com/J-mzz/ddc-safety)

This implementation utilizes the [Casos](https://github.com/iFR-OFC/casos) tool for controller and observer synthesis.
